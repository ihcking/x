# Cloudflare R2 网盘 - 部署指南

> 支持：在线浏览 | 上传文件 | 分享链接 | 下载

---

## 快速部署步骤

### 1. 创建 R2 存储桶

1. 登录 [Cloudflare Dashboard](https://dash.cloudflare.com)
2. 左侧菜单 → **R2** → 创建存储桶
3. 名称：`my-disk`（可以自定义）
4. 创建完成后，点"设置"→"CORS 策略"，添加：
```
[
  {
    "AllowedOrigins": ["*"],
    "AllowedMethods": ["GET", "PUT", "DELETE", "HEAD", "POST"],
    "AllowedHeaders": ["*"]
  }
]
```

### 2. 创建 API Token

1. R2 → 管理 API Token → 创建自定义 Token
2. 权限：
   - 对象读取 (Get, List)
   - 对象写入 (Put, Delete)
   - 对象列出 (List)
3. 创建后复制 Token

### 3. 创建 Worker

1. 左侧菜单 → **Workers 和 Pages** → 创建 Worker
2. 名称：`disk`（可自定义）
3. 点击"编辑代码"，删除默认代码
4. 复制 `cloudflare-disk.js` 的全部内容粘贴进去
5. 点击"保存并部署"

### 4. 绑定 R2 存储桶

1. Worker → 设置 → 变量
2. 添加环境变量：
   - `MY_BUCKET` = 你的存储桶名称（如 my-disk）
3. 添加环境变量（账户 API Token）：
   - `R2_ACCESS_KEY_ID` = 你的 Access Key ID
   - `R2_SECRET_ACCESS_KEY` = 你的 Secret Access Key
4. 点击"保存"

### 5. 配置自定义域名（可选）

1. Worker → 设置 → 触发器 → 添加自定义域名
2. 输入：`disk.yourdomain.com`
3. 开启 Cloudflare CDN

## 功能说明

| 功能 | 说明 |
|------|------|
| 📁 在线浏览 | 访问主页查看所有文件 |
| ⬆️ 上传文件 | 支持任意类型文件上传 |
| ⬇️ 直链下载 | `/d/文件名` 格式 |
| 🔗 分享链接 | 生成24小时有效期链接 |
| 🗑️ 删除文件 | 支持删除操作 |
| 📊 存储统计 | 显示文件数量和用量 |

---

## 文件结构

```
你的网盘/
├── /                     # 首页（文件列表+上传）
├── /upload               # 上传接口（POST）
├── /d/文件名             # 下载直链
├── /s/分享ID             # 分享链接访问
├── /api/delete/文件名    # 删除接口（DELETE）
└── /?action=list        # API: 文件列表
```
